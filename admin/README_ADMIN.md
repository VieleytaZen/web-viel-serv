# 📚 Panduan Penggunaan Panel Admin Viel Digital

Gunakan panel ini untuk mengelola seluruh konten website secara real-time. Berikut adalah panduan singkat agar tidak terjadi kesalahan input:

---

### 🛒 Produk Affiliate (Rekomendasi Setup)
* **Kategori Utama:** Pilih antara Digital, Pakaian, atau Aksesoris.
* **Merk & Sub-Kategori:** Pilih dari daftar yang tersedia. Jika merk/jenis barang tidak ada, ketik manual lalu tekan **Enter**.
* **Rate Produk:** Gunakan angka desimal dengan titik (Contoh: `4.8`).
* **Deskripsi:** Bisa menggunakan format Markdown (seperti membuat list atau teks tebal).

### 💻 Jasa Coding
* **ID (Penting):** Gunakan `web-dev` untuk layanan Landing Page dan `bot-dev` untuk layanan WhatsApp Bot agar ikon di website muncul dengan benar.
* **Detail (HTML):** Kamu bisa menggunakan tag `<br>` untuk baris baru atau `<b>` untuk teks tebal di dalam kolom detail.

### 🧹 Jasa Cleaning
* **Price:** Masukkan format harga lengkap (Contoh: `Rp 50.000 / Jam`).
* **WA Link:** Pastikan link WhatsApp menggunakan format `https://wa.me/62812...`

---
*💡 Tips: Jangan lupa menekan tombol **Save/Publish** di pojok kanan atas setelah melakukan perubahan.*
